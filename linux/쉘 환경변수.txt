[쉘 환경변수 와 앨리어스]


1. 콘쉘

# 절대경로
- /usr/bin/ksh

# 기본프롬프트
- 일반 사용자 : $
- root 사용자 : #

# 환경설정
- 환경설정을 위해 환경변수에 적절한 값을 지정 

# 쉘변수설정

ATEST = test
set 


# 쉘 환경변수

export ATEST = test
env

# 변수 확인

echo $ATEST


# 앨리어스 설정 
- 초기화파일, .kshrc 에 저장하면 로그인시마다 앨리어스가 설정된다.

1.등록하기 
alias ls='ls -F' ls -F 대신 ls를사용
alias c=clear

2.삭제하기
unalias ls 

ls 라는 앨리어스삭제


# 히스토리 기능이란 ?
- 사용자가 사용한 명령 목록을 저장했다 다시사용.

# 히스토리 설정 
- HZ = 200 200개까지 저장가능, 기본값은 100이다.

# 히스토리 저장 
- 로그아웃시 지금까지 입력된 HZ개 명령을 히스토리파일에 저장 


# 이전명령보기 
- 최근 사용한 16개 명령을 번호와 함께 출력 

# 앨리어스이다. 원래명령은 
-ksh93 : hist -l
-ksh88 : fc-l 

# 명령 재실행 
r (recall) 재실행
r : 직전의 명령 재실행
r 64 : 64번째 명령 재실행 

# 방향키 기능 
방향키로 히스토리 목록 출력 


# 파일명 자동완성 기능 
- 파일명 앞 부분을 입력하고 ESC 두번 or 탭키 입력시 자동완성 


# 시스템 초기화파일
/etc/profile 

# 사용자 초기화파일
~/.profile 
- 콘쉘의 모든 사용자에게 기본제공
- 프롬프트,명령어경로등 환경변수설정

~/.kshrc 
-필요시 생성 
-자식쉘에게 적용 


# 터미널 기능설정
stty [옵션] [모드]

옵션 :
-a 터미널 전체옵션 

모드 :
erase : 삭제키 
rows : 터미널의 행수 
columns: 터미널의 열수 

삭제키를 백스페이스로 사용 

stty erase ^H : 백스페이스를 삭제키로 사용 


# 경로 설정 

PATH 환경변수에 지정 

PATH=/bin:/usr/bin:/usr/ucb:/usr/local/bin:

프롬프트 설정 
: PS1에 지정 해야한다. 

PS1 = '[$PWD]' 
[/home/user1] PS1 = '$'

사용자 초기화파일실행
. 명령어를 실행하여 실행 

:. .profile 


2. C 쉘

# 절대경로 
/bin/csh

# 기본프롬프트

일반 : %
루트 : # 

# 환경변수 설정
: setenv[환경변수명 값]

setenv TEAM vt100 할당 
echo $TERM 출력

# 쉘 변수설정
set [쉘 변수명 값]

set term=vt100 
echo $term 

# 앨리어스 설정 

alias 로 설정 
unalias 로 삭제 

alias ls 'ls -alF'
unalias ls

일시적 비활성화 \앨리어스 


# 히스토리기능 

# 히스토리 설정 
- 지역변수형태로 .cshrc 파일에 설정

set history=100 
- 명령 100개 저장가능

# 히스토리 저장 
- 로그아웃후에도 저장 
set savehist=100 
- 홈디렉토리에 .history에 저장된다 . 

# 히스토리 목록보기
: history [옵션]

옵션
: -h 번호생략
: -r 역순
: 숫자 최근실행한 명령 개수 지정 


# 명령 재실행 
: 지정한 문자, 히스토리번호를 갖는 최근명령 실행 
! 문자
! 번호 

# 파일명 자동완성 
- 일부분을 입력하고 ESC를 입력하면 된다.
내장 변수인 filec 를 설정하면 자동완성기능 사용가능.

- set filec 

ls te^D 
: te로 시작하는 모든 파일목록 출력 

# 환경설정파일
- 시스템파일
/etc/login

# 사용자 파일 
~/.login , ~/.cshrc 
로그인시 .cshrc 파일실행후 .login파일이 실행된다. 

.login 
: 로그인시 1회실행 
: 환경변수,터미널설정

.cshrc
: 로그인시,서브쉘생성시
: 쉘변수,앨리어스,히스토리 설정가능.

* 서브쉘 
- 쉘상태에서 다른쉘을 사용가능하다.

.logout
: 로그아웃시 실행 
: 임시파일정리,히스토리파일정리,로그아웃시간기록 


# 초기화파일 실행

:source 초기화파일 

source .chsrc 
source .login
